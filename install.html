🛵 내 작업집계 설치 (안드로이드)

운영웹에서 내 작업을 시간대별로 세고, 결과를 이미지로 카톡 공유까지. 딱 한 번만 등록하면 됩니다.

먼저 아래 버튼으로 코드를 복사하세요
1
크롬에서 아무 사이트나 연 다음, 주소창 옆 ⋮(점 3개) → ☆ 별표를 눌러 북마크에 저장합니다.
2
⋮ → 북마크 → 방금 저장한 북마크를 길게 누르기 → 수정(또는 편집)을 엽니다.
3
이름 칸을 작업집계 로 바꿉니다.
4
URL(주소) 칸의 글자를 전부 지우고, 빈 칸을 길게 눌러 '붙여넣기' → 저장. (1번에서 복사한 코드가 들어갑니다)
붙여넣기가 안 되면, 맨 아래 코드 상자를 길게 눌러 직접 복사해도 됩니다.
5
사용할 때: 운영웹(로그인 상태)에서 크롬 주소창에 작업집계 라고 입력 → 아래 뜨는 '작업집계' 북마크를 탭!

📊 사용법

  • 화면 오른쪽 위에 집계창이 떠요.
  • 야간조(22시~9시) / 주간조(8시~22시) 버튼을 누르거나 시간을 직접 골라 조회.
  • ◀ 하루 전 / 하루 후 ▶ 로 날짜를 하루씩 이동.
  • 이미지로 복사 → 카톡 입력칸 길게 눌러 붙여넣기로 공유.
문제가 생기면 사장님께 알려주세요 🙂
